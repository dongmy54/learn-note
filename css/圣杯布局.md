#### 圣杯布局
> 要求中间内容先渲染

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>圣杯布局</title>
  <style type="text/css">
    *{
      margin: 0;
      padding: 0;
    }

    body{
      min-width: 500px;
    }

    .header, .footer{
      width: 100%;
      height: 50px;
      line-height: 50px;
      text-align: center;
      background: #8b9799;
    }

    .container{
      height: 600px;
      padding: 0 200px 0 220px; /* 给左右侧留出空隙 */
      min-width: 700px; /* 值大于等于 2*左边宽度 + 右边宽度 */
    }

    .left{
      width: 220px;
      background: #ccafc9;
      margin-left: -100%; /* 回到middle占用的位置 */
      position: relative; /* 相对定位再次左靠 */
      left: -220px;
    }

    .middle{
      width: 100%;
      background: green;
    }

    .right{
      width: 200px;
      background: yellow;
      margin-left: -200px; /* 回到中间块右侧 */
      position: relative; /* 相对定位将元素右调 */
      right: -200px;
    }

    .left,.middle,.right{
      float: left;
      min-height: 300px;}

  </style>
</head>
<body>
  <div class="header">
    这是页头
  </div>

  <div class="container">
    <div class="middle">
      这是中间内容
    </div>

    <div class="left">
      这是左侧内容
    </div>

    <div class="right">
      这是右侧内容
    </div>
  </div>

  <div class="footer">
    这是页尾
  </div>
</body>
</html>
```
