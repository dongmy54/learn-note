四

> 关于打印

```go
fmt.Println("a")
// "a" 打印后换行 用于打印单个值
fmt.Println("测试值是：", "hello")
// 多个参数也是可以的

// 用于打印字面变量比较好 还可以切换多种类型
fmt.Printf("%v \n", "a")
fmt.Printf("%v \n", [...]int{1,2,4})
```
