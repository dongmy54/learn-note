### 函数参数

#### 1. 形参与实参数
- 形参：我们定义函数的时候，定义的参数名
- 实参：我们调用函数的时候，传递的真实值

#### 2. 任意参数（只能放在最后一个参数）
任意参数本质上是一个slice

```go
package main

import "fmt"

func main() {
	add(1, 3, 5)
	// sum: 9

	s := []int{1, 3, 5, 8}
	add(s...) // 切片做多参数要用...展开
}

// 任意个int参数
func add(args ...int) {
	sum := 0
	for _, arg := range args {
		sum += arg
	}

	fmt.Printf("sum: %d\n", sum)
}
```

#### 3. 值传递和引用传递
在参数调用的过程中有值传递和引用传递
值传递：值的拷贝（性能取决于对象的大小，函数中不改变参数的值）
引用传递：地址的拷贝（性能较好，在函数中可以改改变传入参数）
