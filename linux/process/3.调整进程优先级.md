### 调整优先级

#### 准备测试shell脚本
> 1. 家目录下创建文件 `a.sh`,内容如下：
> 2. 保存文件后`chmod u+x a.sh` 可执行权限
```bash
#! /bin/bash

# $$ 代表当前进程 id
echo $$

while : # : 代表啥都没有一直循环
do
:
done
```

#### 调整优先级
> 数值越大，优先级越小，占用资源越少

```
./a.sh               # 默认执行优先级为 0
nice -n 15 ./a.sh    # 以15的优先级执行
renice -n 15 2104    # 调整2014进程优先级为 15
```
