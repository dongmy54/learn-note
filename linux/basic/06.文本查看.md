#### 文件查看
> * cat 打印出文件所有内容
> * head 打印文件头部
> * tail 打印文件尾部
> * wc 统计文件行数

#### 用法
```bash
cat temp/demo      # 打印文件所有内容
head temp/demo     #  打印文件前10行
head -3 temp/demo  # 打印文件前3行
tail temp/demo     # 打印文件末尾10
tail -3 temp/demo  # 打印文件末尾3行
tail -f temp/demo  # 实时追踪文件最新信息
wc temp/demo       # 统计文件行数
```
