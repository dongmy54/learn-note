#### vim 使用
> * `esc`进入普通模式(PS：英文输入法)
> * `i` 进入插入模式
> * `:w` 保存
> * `:w ～/a.txt` 保存路径
> * `:q` 退出
> * `:wq` 保存退出
> * `:q!` 不保存退出
> * `:!` 编辑下 查看命令(不同退出编辑器)


##### 使用
```
shift + ^   定位行首（普通模式）
shift + $   定位行尾（普通模式）
shift + i   定位行首                （进入--插入模式）
shift + a   定位行尾                 (进入--插入默认)
a           定位到当前光标下一个字符   （进入--插入模式）

shift + o   当前行的上一行插入空行     （进入--插入模式）
o           _______下一行______      (进入--插入模式)

h j k l    左（h) l(右) j(下) k(上)方向移动 （普通模式）

yy         复制行               (普通模式）
y$         复制当前光标到行末尾    (普通模式）
3yy        复制下3行            （普通模式）

p          粘贴                 (普通模式）

dd         删除行                (普通模式）
d$         删除当前光标到行末尾     (普通模式）
3dd        删除下3行             (普通模式）

u          撤销                 (普通模式）
ctrl + r   重做                （普通模式）

gg         定位行首              (普通模式）
shift + g  定位行尾              (普通模式）
11gg       定位到11行            (普通模式）

x          删除单个字符           (普通模式）
r          修改单个字符           (普通模式）

/hu        查找hu(n 跳到下一个 shift + n跳上一个) (普通模式）

:%s/H/f/g     全局替换（H -> f）       (普通模式）
:3,5s/f/H/g   3-5行全局替换（H -> f）  (普通模式）
:s/f/K        当前行 首个(f -> K)      (普通模式）
:%s/f/K       所有行首个(f -> K)       (普通模式）
PS:
1. % 所有行
2. s 当前行
3. g 整行所有
```
