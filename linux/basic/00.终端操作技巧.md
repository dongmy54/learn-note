#### 惯用
```bash
- 代表上一个目录

上下方向键  # 调出上一条/下一条命令
!!        # 代表上一次命令
!command  # 代表上一次的xx命令

history 10           # 最后10个命令 
history | grep nginx # 默认会有1000个命令存在,这里从命令中过滤
ctrl + R   # 搜索此前命令，会弹出搜索界面，按下enter直接执行、按下左右键，展示在命令行

tab        # 用于命令的补全

ctrl + a    # 回到行首
ctrl + e    # 回到行尾

ctrl + k    # 剪切光标当前位置到行尾内容
ctrl + u    # 剪切光标当前位置当行首内容

clear       # 清屏
```
