用法：`fdisk 硬盘设备`

> 整个过程看似复杂，其实都是一个套路
> 1. 选择新增什么分区

> 3. 指定分区大小

##### step4 挂载
将目录与分区关联起来
```bash
mkdir -p /mnt/sdb          # 创建挂载目录（挂载点）
mount /dev/sdb1 /mnt/b1    # 挂载到 /mnt/b1

mount | grep sdb           # 看当前挂载
umount /dev/sdb1           # 卸载挂载点（可以接设备，也可接挂载点）
```

##### step5 持久化到配置文件
这里持久化的是挂载信息
```bash
vim /etc/fstab  # 配置文件

# 设备名/uuid  挂载位置 格式 权限 磁盘配额
/dev/dsc1 /mnt/dsc ext4 default 0 0
```
