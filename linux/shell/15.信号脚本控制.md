#### 信号脚本控制
> 脚本控制主要有两种方式
> - 通过 renice、nice 命令调整cpu占用
> - 通过发送/捕获信号，控制脚本的停止/保留

##### fork炸弹
```bash
.(){.|.&};& 
# 通过不断的创建进程，达到服务器资源耗尽，不接受任何响应
# 处理办法：强制重启服务器
```

##### 信号
> - kill 默认会发送 15 号信号
> - ctrl + c 发送 2 号信号
> - 9号信号，是不可以阻塞的

```bash
# 捕获15号信号
trap 'echo signal 15' 15 

# 打印出脚本进程
echo $$

# 死循环
while :
do
  :
done

# 将不能杀死该进程，会被捕获
# kill xxx
```
