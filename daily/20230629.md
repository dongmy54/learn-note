#### 1. ruby ENV
```ruby
ENV['a'] # 不存在为nil
=> nil

ENV['b'] = "ab" # 赋值
ENV['b']
=> "ab"

ENV.fetch("foo") # 这样如果foo不存在会报错
ENV.fetch("foo") {"foo"} # 如果没有foo 则返回默认值 推荐用法
```
