##### 创建一个索引包含mapping
```
"number_of_replicas": 0
这里指定副分片数量0，因为在自己单机电脑上

PUT /products
{
  "settings":{
    "number_of_shards": 1,  
    "number_of_replicas": 0 
  },
  "mappings":{ 
      "properties":{
        "name":{ 
          "type": "text",
          "analyzer": "smartcn"
        },
        "brand_name":{
          "type": "keyword"
        },
        "city":{
          "type": "keyword"
        },
        "price":{
          "type": "float"
        },
        "create_at":{
          "type":"date",
          "format":"yyyy-MM-dd HH:mm:ss"
        }
      }
  }
}
```

##### 查看索引
```
GET /products
```

##### 查看mapping
```
GET /products/_mapping
```

##### 更新mapping(添加字段)
```
添加size 类型为integer

PUT /products/_mapping
{
  "properties":{
    "size":{ 
      "type": "integer"
    }
  }
}
```

```
所有数据

  }
}

{

  },

  ],
```
