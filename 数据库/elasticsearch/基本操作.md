##### 创建一个索引包含mapping
```
"number_of_replicas": 0
这里指定副分片数量0，因为在自己单机电脑上

PUT /products
{
  "settings":{
    "number_of_shards": 1,  
    "number_of_replicas": 0 
  },
  "mappings":{ 
      "properties":{
        "name":{ 
          "type": "text",
          "analyzer": "smartcn"
        },
        "brand_name":{
          "type": "keyword"
        },
        "city":{
          "type": "keyword"
        },
        "price":{
          "type": "float"
        },
        "create_at":{
          "type":"date",
          "format":"yyyy-MM-dd HH:mm:ss"
        }
      }
  }
}

常用数据类型
string - 全文检索
keyword - 不可分割做为整体（PS: 不可全文检索）

integer - 整数
float - 浮点数
date - 日期

除了简单的数据类型外，也支持一些复合的数据类型，比如：数组、数组对象等；我们直接写会动态的去创建它
```

##### 查看索引
```
GET /products
```

##### 查看mapping
```
GET /products/_mapping
```

##### 更新mapping(添加字段)
```
添加size 类型为integer

PUT /products/_mapping
{
  "properties":{
    "size":{ 
      "type": "integer"
    }
  }
}
```

##### 索引数据
```
单条索引（不带id-es会自动生成id)

POST products/_doc
{"name": "笔记本-hw-001", "brand_name": "华为", "city": "成都", "price": 2850, "create_at": "2022-03-01 10:12:21"}

批量索引
POST products/_bulk
{ "create": { } }
{"name": "笔记本-hw-001", "brand_name": "苹果", "city": "重庆", "price": 2950, "create_at": "2022-03-01 10:18:21"}

{"name": "笔记本-hw-002", "brand_name": "联想", "city": "北京", "price": 3850, "create_at": "2022-04-01 10:17:21"}
```

##### 更新单个文档
```

```

##### 删除单个文档
```
# 删除文档
DELETE /products/_doc/InDLmIABXDcF4psAxAXV
```

##### 搜索
```
# 名字中包含笔记本的 这也是最基本的

GET products/_search
{
  "query": {
    "match_phrase": {
      "name": "笔记本"
    }
  }
}


# 指定字段、排序、隐藏源数据、限制一条数据
GET products/_search
{
  "query": {
    "match_all": { }
  },
  "fields": [
    "brand_name"
  ],
  "_source": false,
  "size": 1,
  "sort": [
    {
      "created_at": "desc"
    }
  ]
}
```

##### 删除索引
`DELETE products`
